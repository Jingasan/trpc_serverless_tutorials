# Usage

１．Build and deploy

```
$ npm install
$ npm build
$ npm deploy
```

２．Call API

API URL (API Gateway v2):

```
https://<randam>.execute-api.ap-northeast-1.amazonaws.com/trpc/greet
```

API URL (API Gateway v1):

```
https://<randam>.execute-api.ap-northeast-1.amazonaws.com/dev/trpc/greet
```
